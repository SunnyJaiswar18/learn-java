Java if-else Statement

In programming, decision-making is crucial for determining the flow of execution. Whether navigating through various scenarios in a program or controlling program behavior, decisions play an essential role. In Java, the 'if-else' statement is one of the fundamental constructs for decision-making.

The Java if statement is used to test conditions, evaluating a boolean expression as true or false. There are different types of if statements in Java:

  • if statement
  • if-else statement
  • if-else-if ladder
  • Nested if statement

How Does It Work?

When an if-else statement is encountered, the condition is evaluated. If the condition is true, the if block executes. If the condition is false, the else block executes, allowing the program to follow different paths based on the condition’s result.

Java if Statement

The if statement in Java executes a block of code if the condition is true.

Syntax:

if (condition) {
    // code to be executed
}

Example:

//Java Program demonstrating the use of if statement
public class IfExample {
    public static void main(String[] args) {
        int age = 20;
        if (age > 18) {
            System.out.print("Age is greater than 18");
        }
    }
}
// Output: Age is greater than 18

Java if-else Statement

The if-else statement checks a condition, executing the if block if it’s true or the else block if it’s false.

Syntax:

if (condition) {
    // code if condition is true
} else {
    // code if condition is false
}

Example:

//Java Program demonstrating if-else statement (odd/even number check)
public class IfElseExample {
    public static void main(String[] args) {
        int number = 13;
        if (number % 2 == 0) {
            System.out.println("even number");
        } else {
            System.out.println("odd number");
        }
    }
}
// Output: odd number

Leap Year Example Using if-else:

A year is a leap year if it is divisible by 4 and 400, but not by 100.

//Java Program to check for Leap Year
public class LeapYearExample {
    public static void main(String[] args) {
        int year = 2020;
        if (((year % 4 == 0) && (year % 100 != 0)) || (year % 400 == 0)) {
            System.out.println("LEAP YEAR");
        } else {
            System.out.println("COMMON YEAR");
        }
    }
}
// Output: LEAP YEAR

Java if-else-if Ladder Statement

The if-else-if ladder tests multiple conditions sequentially, executing only one of the blocks when a condition is true.

Syntax:

if (condition1) {
    // code if condition1 is true
} else if (condition2) {
    // code if condition2 is true
} else if (condition3) {
    // code if condition3 is true
} else {
    // code if all conditions are false
}

Example:

//Java Program to demonstrate the grading system using if-else-if ladder
public class IfElseIfExample {
    public static void main(String[] args) {
        int marks = 65;

        if (marks < 50) {
            System.out.println("fail");
        } else if (marks >= 50 && marks < 60) {
            System.out.println("D grade");
        } else if (marks >= 60 && marks < 70) {
            System.out.println("C grade");
        } else if (marks >= 70 && marks < 80) {
            System.out.println("B grade");
        } else if (marks >= 80 && marks < 90) {
            System.out.println("A grade");
        } else if (marks >= 90 && marks <= 100) {
            System.out.println("A+ grade");
        } else {
            System.out.println("Invalid!");
        }
    }
}
// Output: C grade

Java Nested if Statement

The nested if statement is an if block within another if block, executed only if the outer condition is true.

Syntax:

if (condition) {
    // code to be executed
    if (condition) {
        // code to be executed
    }
}

Example:

//Java Program to demonstrate Nested If Statement
public class JavaNestedIfExample {
    public static void main(String[] args) {
        int age = 20;
        int weight = 80;

        if (age >= 18) {
            if (weight > 50) {
                System.out.println("You are eligible to donate blood");
            }
        }
    }
}
// Output: You are eligible to donate blood

Ternary Operator

The ternary operator (? :) is a shorthand way to perform the task of an if-else statement. It evaluates a condition and returns one of two values.

Example:

//Java Program demonstrating ternary operator
public class TernaryExample {
    public static void main(String[] args) {
        int number = 13;
        String output = (number % 2 == 0) ? "even number" : "odd number";
        System.out.println(output);
    }
}
// Output: odd number

Conclusion

The if-else statement is a vital tool in Java programming, enabling dynamic decision-making and diverse execution paths based on conditions. By mastering its usage, developers can create more responsive and robust applications.

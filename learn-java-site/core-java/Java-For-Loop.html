Java Loops

The Java for loop is used to iterate a part of the program several times. If the number of iterations is fixed, it is recommended to use the for loop.

There are three types of for loops in Java:

  • Simple for Loop
  • For-each or Enhanced for Loop
  • Labeled for Loop

Java Simple For Loop

A simple for loop is similar to C/C++. It initializes a variable, checks the condition, and increments/decrements the value. It consists of four parts:

  • Initialization: Executed once when the loop starts.
  • Condition: Evaluated each time to check if the loop continues.
  • Increment/Decrement: Increases or decreases the variable value.
  • Statement: Code executed each time the condition is true.

for(int i=1;i<=10;i++){
    System.out.println(i);
}

Output:

1
2
3
4
5
6

10

Java Nested For Loop

If one for loop is inside another, it is known as a nested for loop. The inner loop executes completely whenever the outer loop executes.

for(int i=1;i<=3;i++){
    for(int j=1;j<=3;j++){
        System.out.println(i+" "+j);
    }
}

Output:

1 1
1 2
1 3
2 1
2 2
2 3
3 1
3 2
3 3

Java For-Each Loop

The for-each loop is used to traverse arrays or collections in Java. It is easier to use than a simple for loop because you don't need to increment the value or use subscript notation.

int arr[]={12,23,44,56,78};
for(int i : arr){
    System.out.println(i);
}

Output:

12
23
44
56
78

Java Labeled For Loop

In a labeled for loop, we give a name to each loop. This is useful in nested loops where we can break or continue a specific loop.

aa:
    for(int i=1;i<=3;i++){
        bb:
        for(int j=1;j<=3;j++){
            if(i==2 && j==2){
                break aa; // Breaks outer loop
            }
            System.out.println(i+" "+j);
        }
    }

Output:

1 1
1 2
1 3
2 1

Java Infinite For Loop

An infinite for loop can be created by using no conditions in the for loop.

for(;;){
    System.out.println("infinitive loop");
}

Output:

infinitive loop
infinitive loop
infinitive loop

Press Ctrl+C to stop the infinite loop.

Loop TypeWhen to UseSyntaxExample

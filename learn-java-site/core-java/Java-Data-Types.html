Data Types in Java

Data types specify the different sizes and values that can be stored in the variable. There are two types of data types in Java:

  • Primitive data types: The primitive data types include boolean, char, byte, short, int, long, float, and double.
  • Non-primitive data types: The non-primitive data types include Classes, Interfaces, and Arrays.

Java Primitive Data Types

In Java language, primitive data types are the building blocks of data manipulation. These are the most basic data types available in Java language.

Java is a statically-typed programming language, meaning all variables must be declared before use. That is why we need to declare a variable's type and name.

There are 8 types of primitive data types:

  • boolean data type
  • byte data type
  • char data type
  • short data type
  • int data type
  • long data type
  • float data type
  • double data type

Boolean Data Type

The Boolean data type is used to store only two possible values: true and false. This data type is used for simple flags that track true/false conditions.

    Boolean one = false;

Byte Data Type

The byte data type is an 8-bit signed two's complement integer with a value range of -128 to 127. Its default value is 0. It is used to save memory in large arrays.

    byte a = 10;
    byte b = -20;

Short Data Type

The short data type is a 16-bit signed two's complement integer with a value range of -32,768 to 32,767. It can also be used to save memory, being smaller than an integer.

    short s = 10000;
    short r = -5000;

Int Data Type

The int data type is a 32-bit signed two's complement integer with a value range from -2^31 to 2^31 - 1. Its default value is 0, and it is generally used as the default data type for integral values.

    int a = 100000;
    int b = -200000;

Long Data Type

The long data type is a 64-bit signed two's complement integer with a value range from -2^63 to 2^63 - 1. It is used when a wider range than int is needed.

    long a = 100000L;
    long b = -200000L;

Float Data Type

The float data type is a single-precision 32-bit IEEE 754 floating point. It is used to save memory in large arrays of floating point numbers but is not recommended for precise values like currency.

    float f1 = 234.5f;

Double Data Type

The double data type is a double-precision 64-bit IEEE 754 floating point and is generally used for decimal values.

    double d1 = 12.3;

Char Data Type

The char data type is a single 16-bit Unicode character. Its value-range is from '\u0000' to '\uffff'. The char data type is used to store characters.

    char letterA = 'A';

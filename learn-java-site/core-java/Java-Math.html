Java Math

The Math class in Java provides methods for performing basic numeric operations like finding the absolute value, exponentiation, square roots, and trigonometric functions.

double result = Math.sqrt(25);  // Returns 5.0

Math Methods

  • Math.abs(double a) - Returns the absolute value of a.
  • Math.pow(double a, double b) - Returns a raised to the power of b.
  • Math.sqrt(double a) - Returns the square root of a.
  • Math.max(a, b) and Math.min(a, b) - Returns the maximum/minimum value of two numbers.
  • Math.random() - Returns a random double between 0.0 and 1.0.

    int max = Math.max(5, 10);       // 10
    int min = Math.min(5, 10);       // 5
    double power = Math.pow(2, 3);   // 8.0
    double random = Math.random();   // Random number between 0.0 and 1.0

Example: Generating a Random Number

To generate a random integer between 1 and 100:

    int randomNum = (int)(Math.random() * 100) + 1;
    System.out.println(randomNum);

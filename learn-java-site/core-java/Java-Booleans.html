Java Booleans

Booleans in Java can have two values: true or false. They are primarily used in control statements for decision-making.

    boolean isJavaFun = true;
    boolean isFishTasty = false;

Example: Using Booleans in Conditional Statements

Booleans are often used in if statements:

    if (isJavaFun) {
        System.out.println("Java is fun!");
    } else {
        System.out.println("Java is not fun.");
    }
